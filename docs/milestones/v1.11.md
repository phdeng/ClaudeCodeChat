# v1.11.0 — 消息气泡样式优化 + 快速笔记

## 功能清单

### 1. 消息气泡样式优化
- **描述**：用户消息和 AI 消息视觉区分更明显
- **用户消息**：右对齐 + 主题色实心气泡（`bg-primary`）+ 白色文字 + 轻阴影
- **AI 消息**：左对齐 + 淡灰半透明卡片背景（`bg-muted/40`）+ 圆角
- **代码块兼容**：用户气泡内的代码块通过 CSS 选择器覆盖颜色，保持可读性
- **响应式**：移动端气泡宽度 90%，桌面端 80%
- **文件**：修改 `components/MessageList.tsx`

### 2. 快速笔记 Scratch Pad
- **描述**：浮动便签窗口，用于临时存放想法、代码片段、提示词草稿，不发送给 Claude
- **交互**：
  - 聊天页右下角浮动按钮（StickyNote 图标）
  - 点击弹出 320x400 浮动面板
  - 顶部标题栏 + 最小化/关闭按钮
  - 中间 textarea 编辑区
  - 底部工具栏：字数统计 + 复制按钮 + 清空按钮
  - 内容非空时浮动按钮显示蓝色圆点指示
  - localStorage 持久化内容
- **文件**：
  - 新建 `components/ScratchPad.tsx`
  - 新建 `stores/scratchPadStore.ts`
  - 修改 `pages/ChatPage.tsx`
