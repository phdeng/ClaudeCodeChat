# v1.8.0 — 智能交互与开发者体验

## 功能清单

### 1. 内嵌终端面板
- **描述**：在文件浏览器下方或独立面板中嵌入 xterm.js 终端，支持直接在 Web UI 中执行命令
- **交互**：焦点模式下，文件浏览器底部可拖拽分隔出终端区域
- **文件**：新建 `TerminalPanel.tsx`、后端新增 WebSocket 终端代理端点
- **复杂度**：高

### 2. 消息收藏夹增强
- **描述**：跨会话收藏消息，独立面板浏览、搜索、分类管理
- **交互**：消息上下文菜单添加"收藏"选项，侧边栏新增收藏夹入口
- **文件**：新建 `favoritesStore.ts`、新建 `FavoritesPanel.tsx`、修改 `MessageList.tsx`
- **复杂度**：中

### 3. AI 对话摘要
- **描述**：对长对话自动生成摘要，帮助快速回顾对话要点
- **交互**：对话标题栏新增"生成摘要"按钮，摘要显示在对话顶部可折叠区域
- **文件**：修改 `ChatPage.tsx`、新建 `ConversationSummary.tsx`
- **复杂度**：中

### 4. 智能输入自动补全
- **描述**：输入 @ 时弹出文件路径补全，输入 / 时弹出命令补全，支持模糊匹配
- **交互**：下拉补全列表，键盘上下选择 + Enter 确认 + Esc 取消
- **文件**：新建 `AutoComplete.tsx`、修改 `ChatInput.tsx`
- **复杂度**：中
