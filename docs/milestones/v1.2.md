# v1.2 里程碑 — 国际化 + 使用统计 + 工作流 + 知识库

## 版本目标

实现完整的中英文国际化支持，增加使用统计仪表板、CLI 输出结构化解析、提示词工作流编排、知识库管理五大功能。

## 功能清单

### 国际化 (i18n)
- [x] 中文语言包（zh.ts，320+ 翻译键值）
- [x] 英文语言包（en.ts，完整对照翻译）
- [x] i18n 基础设施（useTranslation hook + Zustand 持久化）
- [x] `DeepString<T>` 递归泛型解决类型兼容
- [x] 全部界面组件替换为 `t()` 调用（15+ 文件）
- [x] 设置页「界面语言」切换卡片
- [x] 语音识别语言自动切换（zh-CN / en-US）

### 使用统计仪表板
- [x] 4 概览卡片（总 Token / 估算成本 / 总对话 / 平均每次）
- [x] SVG 手绘柱状图（14 天趋势，input/output 双色段）
- [x] 模型使用分布条形图
- [x] 项目 Top-5 排行
- [x] Token 单价参考表
- [x] 设置页新增「使用统计」Tab

### CLI 工具输出解析
- [x] 7 种工具专用渲染器：
  - Edit — Diff 对比视图
  - Bash — 终端风格输出
  - Glob/Grep — 文件列表 + 复制按钮
  - Read — 代码块渲染
  - LS — 树形目录结构
  - MultiEdit — 批量编辑展示
  - Generic — 通用 JSON 回退
- [x] 替代原始 JSON 展示，可读性大幅提升

### 提示词工作流编排
- [x] 工作流 CRUD（名称 / 描述 / 多步骤）
- [x] `{{prev}}` 变量替换（引用上一步 AI 回复）
- [x] 步骤拖拽排序（上移/下移）
- [x] 链式自动执行（SSE done 事件触发步进，500ms 延迟）
- [x] 3 个预置模板（代码审查 / TDD / 文档生成）
- [x] 执行进度条（WorkflowExecutionBar）
- [x] `/workflow` 命令面板入口

### 知识库
- [x] 消息保存到知识库（SaveToKnowledgeDialog）
- [x] 自动提取标题 + 标签输入
- [x] 全文搜索 + 标签筛选
- [x] Markdown 详情预览（react-markdown + GFM + KaTeX）
- [x] 独立页面路由（/knowledge，懒加载）
- [x] 移动端底部导航入口
- [x] Zustand persist 持久化
