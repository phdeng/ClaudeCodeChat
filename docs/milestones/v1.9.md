# v1.9.0 — MCP 市场 + Skill 市场

## 功能清单

### 1. 市场模块（独立页面 /marketplace）
- 独立路由 `/marketplace`，懒加载
- 顶部 Tab 切换：MCP 市场 / Skill 市场
- 搜索栏 + 分类筛选
- 侧边栏 + 底部导航栏增加市场入口

### 2. MCP 市场
- 预置 12+ 热门 MCP 服务器（从 modelcontextprotocol 官方仓库精选）
- 每项显示：名称、图标、描述、作者、分类标签
- "添加到设置" 按钮 → 调用 POST /api/config/mcp-servers 直接安装
- 已安装状态检测（与设置页联动）
- 分类：文件系统、数据库、搜索、开发工具、通信、AI增强

### 3. Skill 市场
- 预置 8+ 实用 Skill 模板
- 每项显示：名称、图标、描述、SKILL.md 内容预览
- "添加到项目" 按钮 → 调用 POST /api/config/skills 安装到当前项目
- 分类：代码质量、开发流程、文档、测试、安全

### 4. 数据结构

MCP 市场项：
```typescript
interface McpMarketItem {
  id: string
  name: string           // 服务器名称（安装后的 key）
  displayName: string    // 显示名称
  description: string
  category: string
  icon: string           // lucide 图标名
  author: string
  config: {
    command: string
    args: string[]
    env?: Record<string, string>
  }
  tags: string[]
  popularity: number     // 热度排序
}
```

Skill 市场项：
```typescript
interface SkillMarketItem {
  id: string
  name: string           // skill 目录名
  displayName: string
  description: string
  category: string
  icon: string
  author: string
  skillMd: string        // SKILL.md 内容
  tags: string[]
  popularity: number
}
```
