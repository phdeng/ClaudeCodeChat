# v1.6 里程碑 — 文件浏览器 + 聚焦分屏模式

## 版本目标

将文件浏览功能从 /context 面板迁移到对话页面主界面，聚焦模式下启用左右分屏布局（文件浏览器在左，对话在右），支持可拖拽调节分屏比例。

## 技术方案

### 1. zenMode 状态迁移
```typescript
// settingsStore.ts 新增
zenMode: boolean
toggleZenMode: () => void
setZenMode: (value: boolean) => void
```
ChatLayout.tsx 改为 `const { zenMode, toggleZenMode, setZenMode } = useSettingsStore()`。

### 2. 分屏布局（聚焦模式）
```
┌─────────────────────────────────────────────┐
│ zen-toolbar (浮动工具条)                     │
├──────────────┬──────────────────────────────┤
│ FileExplorer │ main (ChatPage)              │
│ (可拖拽宽度)  │                              │
│              │                              │
│ 面包屑导航    │ MessageList                  │
│ 搜索框       │                              │
│ 文件列表     │ ChatInput                    │
├──────────────┴──────────────────────────────┤
```

### 3. 拖拽分隔条
- CSS: `cursor: col-resize`, 4px 宽度
- mousedown → mousemove 计算新宽度 → mouseup 停止
- 最小 200px，最大 50% 视口宽度
- 宽度持久化到 fileExplorerStore (localStorage)

### 4. `/files` 命令
ChatPage.tsx 的 directCommands 新增 `/files`：
- 调用 `setZenMode(true)` 进入聚焦模式
- 调用 `setShowFileExplorer(true)` 打开文件浏览器
