# v1.3 里程碑 — /context 上下文可视化 + 安全防护 + 成本控制

## 版本目标

实现 CLI `/context` 命令的 Web 可视化（彩色网格 + Token 类别分解 + MCP/Skills 展示），增加文件预览、敏感信息检测、成本预警等安全与效率功能。

## 发布信息

- **版本号**: 1.3.0
- **发布日期**: 2026-03-12
- **Git 标签**: `v1.3.0`
- **分支**: `release/v1.3.0`

## 功能清单

### /context 上下文可视化（P0）
- [x] 彩色方块网格（10x10 grid）可视化上下文占用分布
- [x] Token 类别分解：System prompt / System tools / Memory files / Skills / Messages / User context / Free space / Autocompact buffer
- [x] 模型信息 + 总量/已用/百分比显示
- [x] MCP tools 列表（从 /api/config/mcp-servers 获取）
- [x] Skills 列表（从 /api/config/skills 获取）
- [x] "发送 /context 到 CLI" 按钮获取精确数据
- [x] 双 Tab 切换：Context Usage 可视化 + 管理上下文（文件/目录/URL）
- [x] 上下文 Store（Zustand persist）管理会话级上下文项

### 文件预览与内容查看器（P0）
- [x] 文件内容预览弹窗（代码高亮 + 行号 + 图片 + Markdown）
- [x] 后端 API：GET /api/filesystem/file-info（文件元信息）
- [x] 后端 API：GET /api/filesystem/read-file（文件内容读取 + 截断 + 二进制检测）
- [x] 路径安全检查（禁止 .. 路径遍历）
- [x] "添加到上下文" 按钮

### 敏感信息自动检测与遮罩（P1）
- [x] 7 类敏感信息正则检测：API Key / 密码 / 私钥 / 手机号 / 身份证号 / Token / 数据库连接串
- [x] 发送消息前自动扫描
- [x] 警告弹窗：三种选择（遮罩后发送 / 确认原文发送 / 取消）
- [x] maskSensitive 函数：保留前3后3，中间 *** 替换

### 智能成本预警与预算控制（P1）
- [x] BudgetAlert 组件：Token 消耗达到预算阈值时显示黄色/红色警告条
- [x] 设置页「成本预算」卡片：单会话预算 + 预警阈值（50%-100%）
- [x] localStorage 持久化预算配置
- [x] 进度条 + 消耗百分比 + 可关闭

### 侧边栏清理
- [x] 清理 DropdownMenu 方案遗留的无用状态变量和 useEffect
