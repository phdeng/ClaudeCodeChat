# v1.10.0 — 侧边栏底部精简 + 交互增强

## 功能清单

### 1. 侧边栏底部工具栏精简（核心需求）
- **问题**：底部 6 个按钮竖排占用 ~250px，挤压历史会话区域
- **方案**：改为单行水平图标栏（~40px 高度），每个按钮只显示图标 + Tooltip
- **布局**：`flex justify-around` 水平排列，图标 16px，间距均匀
- **按钮**：会话历史 / 消息收藏 / 知识库 / 插件市场 / 设置 / 导入
- **收藏 badge**：保留数量徽标在图标右上角
- **文件**：`Sidebar.tsx`

### 2. 对话标签页（多会话并行）
- **描述**：聊天区域顶部支持多个标签页，可同时打开多个会话，快速切换
- **交互**：点击侧边栏会话在新标签页打开，标签页可关闭/拖拽排序
- **文件**：新建 `SessionTabs.tsx`、修改 `ChatLayout.tsx`、修改 `ChatPage.tsx`

### 3. 消息气泡样式优化
- **描述**：用户消息和 AI 消息视觉区分更明显，用户消息右对齐+彩色气泡
- **文件**：修改 `MessageList.tsx` 样式

### 4. 输入框增强 — 多行自适应 + 字数统计
- **描述**：输入框自适应高度（最高 200px），底部显示字数统计
- **文件**：修改 `ChatInput.tsx`
