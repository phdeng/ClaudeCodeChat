# v1.7.0 — 交互增强与效率提升

## 功能清单

### 1. 侧边栏 DropdownMenu 重构
- **问题**：悬浮 7 个操作按钮遮挡会话标题，视觉混乱
- **方案**：替换为单个 `⋯` 按钮 + DropdownMenu 下拉菜单
- **文件**：`Sidebar.tsx`、新建 `dropdown-menu.tsx`
- **复杂度**：中

### 2. 拖拽文件到对话
- **描述**：将文件拖拽到聊天输入区域，自动添加为 `@文件路径` 引用
- **交互**：拖拽时显示蓝色虚线边框提示区域，松开后插入文件路径
- **文件**：`ChatInput.tsx`、`ChatPage.tsx`
- **复杂度**：低

### 3. 提示词模板库
- **描述**：预设常用提示词模板（代码审查、Bug修复、重构、解释代码等），一键填入输入框
- **交互**：输入框上方工具栏新增模板按钮，点击展开模板列表
- **存储**：Zustand store 持久化，支持自定义模板 CRUD
- **文件**：新建 `promptTemplateStore.ts`、新建 `PromptTemplates.tsx`、修改 `ChatInput.tsx`
- **复杂度**：中

### 4. 会话内消息搜索
- **描述**：Ctrl+F 在当前会话消息中搜索关键词，高亮匹配并跳转定位
- **交互**：顶部浮动搜索栏，显示匹配数量，上/下箭头切换匹配项
- **文件**：新建 `MessageSearch.tsx`、修改 `ChatPage.tsx`、修改 `MessageList.tsx`
- **复杂度**：中

## 验证
1. `pnpm build` 构建通过
2. 侧边栏悬浮只显示 `⋯` 按钮，下拉菜单所有操作可用
3. 拖拽文件到聊天区域，正确插入 @文件引用
4. 模板库可展开、选择、自定义模板
5. Ctrl+F 搜索高亮 + 上下切换 + 跳转定位
