# v1.12.0 — 快速会话切换 + 输入框统计

## 功能清单

### 1. 快速会话切换器 (Ctrl+Shift+H)
- **描述**：类似 VS Code 文件切换的快速导航面板，支持模糊搜索会话标题
- **交互**：
  - Ctrl+Shift+H 打开浮动面板（居中 Dialog）
  - 显示最近 10 个活跃会话
  - 输入关键词模糊搜索过滤
  - 上下箭头导航 + Enter 切换 + Esc 关闭
  - 显示会话标题、最后消息时间、消息数
- **文件**：
  - 新建 `components/QuickSessionSwitcher.tsx`
  - 修改 `layouts/ChatLayout.tsx` — 快捷键监听 + 渲染

### 2. 输入框字数/Token 实时统计
- **描述**：输入框下方显示实时字数和预估 Token 数
- **交互**：
  - 输入框底部右侧显示 `XX 字 | ~YY tokens`
  - 灰色小字，不干扰输入体验
  - 超过阈值（如 2000 tokens）时变橙色提示
- **文件**：修改 `components/ChatInput.tsx`（已有 estimateTokens 函数）
