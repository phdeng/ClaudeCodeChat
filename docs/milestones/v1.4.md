# v1.4 里程碑 — CLI 深度参数 + Git 操作 + 自动摘要

## 版本目标

对接 Claude CLI 未暴露的高级参数（effort/budget/fallback/tools），构建 Git 操作集成面板，实现对话自动摘要功能。

## 发布信息

- **版本号**: 1.4.0
- **发布日期**: 2026-03-12
- **Git 标签**: `v1.4.0`
- **分支**: `release/v1.4.0`

## 功能清单

### CLI 深度参数面板（P0）
- [x] Effort 推理深度（4 档：low/medium/high/max → `--effort`）
- [x] 预算硬限制（`--max-budget-usd`，CLI 层面费用上限）
- [x] Fallback 模型（`--fallback-model`，主模型过载时自动降级）
- [x] 工具权限精细控制（`--allowedTools` / `--disallowedTools` 白/黑名单）
- [x] PR 关联（`--from-pr` 参数支持）
- [x] 顶部工具栏高级参数按钮（SlidersHorizontal 图标 + Popover 面板）
- [x] 非默认 effort 时显示蓝色指示圆点
- [x] 参数实时保存到 Session + 发送时传递给后端

### Git 操作集成面板（P0）
- [x] Git 状态可视化：分支名 + ahead/behind + staged/modified/untracked 文件分组
- [x] Diff 预览：点击文件展开 unified diff（绿色增/红色删/蓝色 @@ 头）
- [x] 快捷操作：一键让 Claude commit / review / push
- [x] 后端 API：`GET /api/filesystem/git-status`（解析 porcelain 格式）
- [x] 后端 API：`GET /api/filesystem/git-diff`（文件级 diff + 统计）
- [x] /git 命令升级为打开 Git 面板（替代原文本输出）

### 对话自动摘要（P1）
- [x] 后端 `POST /api/chat/summarize`：使用 Claude Haiku 生成 1-2 句摘要 + 关键词
- [x] SSE done 事件后异步触发（消息 >= 4 条时）
- [x] 摘要和关键词存储到 Session（summary + keyTopics）
- [x] 通过 stdin 传入 prompt（避免 Windows 引号问题）
- [x] 失败静默处理，不阻塞用户操作

### Session 模型扩展
- [x] Session 接口新增：effort / maxBudgetUsd / fallbackModel / allowedTools / disallowedTools / summary / keyTopics
- [x] 6 个 setter 方法 + setSessionSummary 支持同时更新 keyTopics
