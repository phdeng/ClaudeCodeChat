# v1.5 里程碑 — 性能优化 + 全局搜索增强 + 对话分支可视化

## 版本目标

拆分 4200+ 行的 MessageList 巨型组件为独立模块，引入虚拟滚动提升长对话性能；构建全局搜索增强面板支持多维度筛选；实现对话分支可视化展示 fork 关系。

## 发布信息

- **版本号**: 1.5.0
- **发布日期**: 2026-03-12
- **Git 标签**: `v1.5.0`
- **分支**: `release/v1.5.0`

## 功能清单

### 性能优化 — MessageList 拆分 + 虚拟滚动（P0）
- [ ] MessageList.tsx 拆分为 4 个独立模块：MessageList（容器）、MessageItem（单条消息）、MessageActions（操作按钮）、MessageContent（内容渲染）
- [ ] React.memo 包裹 MessageItem 避免无关消息重渲染
- [ ] 引入 react-virtuoso 虚拟滚动，仅渲染可视区域消息
- [ ] 代码高亮懒加载（IntersectionObserver 触发）
- [ ] 自动滚动到底部 + 新消息提示

### 全局搜索增强面板（P0）
- [ ] 新建 GlobalSearchPanel 组件（Sheet 抽屉）
- [ ] 多维度筛选：日期范围、标签、颜色标签、项目路径、消息角色
- [ ] 搜索结果按会话分组 + 上下文片段高亮
- [ ] 点击结果跳转到对应会话的对应消息
- [ ] 搜索历史（最近 10 条，localStorage 持久化）

### 对话分支可视化（P1）
- [ ] Session 接口扩展：parentSessionId + forkFromMessageIndex
- [ ] forkSession 方法记录父子关系
- [ ] SessionTreeView 组件：SVG/CSS 树形展示分支关系
- [ ] 消息级分叉标记（GitBranch 图标 + tooltip）
